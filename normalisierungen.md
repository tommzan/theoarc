# Normalisierungen
- lupiter --> Iupiter
- luno --> Iuno
- love / loue --> Iove
- & --> et
- saturn --> Saturn
- jus --> ius
- siue --> sive
- ceres --> Ceres
- vt --> ut
- vr --> ur
- vn --> un
- vm --> um
- vl --> ul
- vb --> ub
- ē / è / é / ë --> e
- à / á / â --> a
- ô / ó / ò --> o
- û / ú/ ù --> u
- ÿ / ȳ / y --> ij

Bei der Theosophia wurde das "ê" zu einem "est" aufgelöst.
